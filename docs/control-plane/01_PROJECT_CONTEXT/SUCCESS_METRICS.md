# Success Metrics

## Tenant outcomes (product-level)
- Fewer missed deadlines (deadline awareness + reminders)
- Better organized evidence (upload → categorize → cite)
- Higher-quality first drafts of motions/answers
- Increased attorney adoption (usable packets)

## System outcomes (engineering-level)
- Deterministic regeneration (same notebook → same output class)
- Traceability (every claim ties to evidence IDs)
- Low hallucination rate (measured by uncited assertions)
- Fast onboarding of new agents/models (no re-teaching)

## UX outcomes
- Users can find any document in <= 2 clicks
- Clear “what’s next” guidance at all times
